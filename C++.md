## 第一个C++程序

**创建项目：**Visual Studio 2017 -> 创建新项目 -> Visual C++ -> 空项目 -> 项目名称（解决方案名称 与 项目名称 一样即可）

**创建文件：**源文件 -> 右键 -> 添加 -> 新建项 -> C++文件 -> 名称 -> 添加

**编写代码：**

```c++
#include <iostream>
using namespace std;

int main()
{
  cout << "Hello world!" << endl;
  system("pause");
  return 0;
}
```

**运行程序：**点击“本地windows调试”
